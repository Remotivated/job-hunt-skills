# Prompt Library

These prompts work in ChatGPT, Gemini, Claude.ai, or any LLM where you can paste your own materials. They are the simplest way to use Job Hunt Skills without installing a plugin.

## How To Use Them

1. Pick one prompt for the task you are doing.
2. Paste the prompt into your LLM.
3. Replace the bracketed input notes with your resume, job posting, LinkedIn copy, company notes, or other source material.
4. Review the output before sending anything.

The rule is simple: do not let the model invent for you. If it adds a metric, tool, title, employer, project, result, or story detail that you did not provide, verify it or remove it.

If your source material is thin, ask the model to pause and ask targeted questions before drafting. A slower first answer is better than polished fiction.

## What Prompts Cannot Do

Prompts cannot read saved files, remember your story bank, generate DOCX/PDF files, or check claims against a local evidence folder. A prompt can compare a draft against source material you paste in, but it cannot verify facts you did not provide. The Claude Code and Cowork skills can do more because they can work with files.

For prompt-only use, the final review is yours. Treat `[ASK: ...]` and `[VERIFY: ...]` markers as blockers, not suggestions.
