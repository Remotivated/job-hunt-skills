# Changelog

## 1.0.0 - 2026-04-28

Initial shippable release of Job Hunt Skills.

- Hard-renamed the project from `job-hunt-os` to `job-hunt-skills`.
- Updated plugin/package metadata, install commands, and public namespace.
- Repositioned the toolkit for general jobseekers, with remote/hybrid company judgment as a specialty.
- Replaced `remote-culture-check` with `company-research`.
- Added first-class prompt-library documentation for ChatGPT, Gemini, Claude.ai, and other LLM users.
- Removed launch/promo drafts from the tracked repo.
- Added contributor guidance, expanded content hygiene checks, and CI.

Breaking change: the old `job-hunt-os` plugin namespace is not preserved.
